This page shows how to install and use Antrea CNI plugin on Kubernetes.
For background on Project Antrea, read the [Introduction to Antrea](httpsantrea.iodocs).

# #  heading prerequisites

You need to have a Kubernetes cluster. Follow the
[kubeadm getting started guide](docsreferencesetup-toolskubeadm) to bootstrap one.

# # Deploying Antrea with kubeadm

Follow [Getting Started](httpsgithub.comvmware-tanzuantreablobmaindocsgetting-started.md) guide to deploy Antrea for kubeadm.

# #  heading whatsnext

Once your cluster is running, you can follow the [Declare Network Policy](docstasksadminister-clusterdeclare-network-policy) to try out Kubernetes NetworkPolicy.
